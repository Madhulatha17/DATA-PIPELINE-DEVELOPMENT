# DATA-PIPELINE-DEVELOPMENT


## Project Description

This project demonstrates the development of an automated ETL (Extract, Transform, Load) pipeline using Python, Pandas, and Scikit-Learn.

## Objective

To automate data preprocessing, transformation, and loading operations for efficient data preparation and analysis.

## Features

* Data Extraction from CSV files
* Missing Value Handling
* Data Transformation
* Feature Scaling using StandardScaler
* Automated Data Loading
* ETL Workflow Automation

## Technologies Used

* Python
* Pandas
* Scikit-Learn

## Project Workflow

### Extract

Loads raw data from a CSV file.

### Transform

* Removes missing values
* Identifies numeric features
* Applies feature scaling

### Load

Stores the processed dataset into a new CSV file.

## Installation

pip install -r requirements.txt

## Execution

python data_pipeline.py

## Deliverable

A Python-based ETL pipeline automating data preprocessing, transformation, and loading operations.
